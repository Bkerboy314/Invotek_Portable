# Invotek CSS Sheet (Portable)
A portable version of the CSS code used for the VotV archive.
To use it, paste this line of code in your website

```
<link rel="stylesheet" href="styles.css">
```

To see what's new, check out the [Release Notes](https://github.com/Bkerboy314/Invotek_Portable/releases).

For bug reports or feature requests, visit the [Issues](https://github.com/Bkerboy314/Invotek_Portable/issues) page.
